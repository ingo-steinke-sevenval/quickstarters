# Grav

Grav CMS https://getgrav.org/

Grav is a free and open source file-based content management system.

## Start it

```
sloppy start -var=URI:mydomain.sloppy.zone grav.yml
   
Example:
   
sloppy start -var=URI:myplone.sloppy.zone grav.yml
```
